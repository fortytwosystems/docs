FortyTwo CAN
--------------------
.. image:: https://img.shields.io/badge/version-1.0.0-brightgreen

https://github.com/fortytwosystems/ArduinoLibrary-CAN

FortyTwo CAN is an Arduino library for connecting the SAMC21 microcontrollers to a CAN bus.

It is modified from https://github.com/hugllc/samc21_can, which is developed from https://github.com/coryjfowler/MCP_CAN_lib.
